# ASAK Admin React Mock 구현 계획 (2026-07-14 ~ 2026-07-22)

> 목표: 7월 22일까지 관리자 운영 화면을 실제 서버 없이 React + mock 데이터로 시연 가능하게 만든다. 주문 키오스크 화면(`SCR-001~008`, `SCR-012`)은 `ASAK-kiosk`의 범위다.

## 0. 범위와 완료 기준

| 구분 | 화면 | React mock 완료 범위 |
| --- | --- | --- |
| 로그인 | `SCR-015` | 성공/실패, mock 세션, 비인증 접근 차단 |
| 주문 운영 | `SCR-009`, `SCR-010` | 목록/필터, 상세, 상태 변경 |
| 품절 운영 | `SCR-011` | 메뉴·재료·옵션 검색/필터, 토글과 실패 복구 |
| 메뉴 운영 | `SCR-016`, `SCR-017` | 목록, 등록/수정 폼, 기본 검증 |
| 결제수단 운영 | `SCR-018` | 목록, 활성화, 노출 순서 mock |
| 매출 조회 | `SCR-019` | 기간 선택, 일별 매출, 인기 메뉴 요약 |

실제 인증, 실제 결제, 실제 서버 연결, 실시간 알림은 이번 기간에 구현하지 않는다. 화면별 success/loading/empty/error, mock 성공·실패 흐름, `npm run lint`, `npm run build`를 충족해야 완료다.

├─ feature/admin-auth-orders-scr009-010-015
